  • 신상 미리보기 방문은 많지만 구매 불가 구조

데이터 기반 신규 배너 결정

첫 번째 분석에서 구매율이 가장 높은 테마는 패밀리룩(0.58%). 목적이 명확한 고객이 꼼꼼히 탐색 후 구매하는 패턴.

신규 6번 배너도 목적이 명확한 상황을 잡아야 구매로 이어질 것으로 예상.

→ 신규 배너 확정: "선물 고민 끝" (출산/돌 선물 세트 큐레이션)

근거: 선물 구매자는 목적이 명확 + 세트 구매(평균 결제 34,654원) + 상시 수요. "첫 출산준비"와는 타겟이 다름 (본인 구매 vs 선물 구매)

참고: 메인 큐레이션 성과 (2026.03.24 기준) ▸

65개 제품 분석 — 평균 매출 변화 +75.9%, 총 매출 증가 +1,260만원

잘 팔린 제품: 라온 세트(+1,690%), 슈슈 리본보넷(+800%), 스무드 바디수트(+569%)

세트 상품 평균 결제 34,654원 vs 단품 19,348원 (1.8배)

배치 효과: 대부분 8~14일 후 약해짐 → 10일 단위 교체 권장

현재 배너의 핵심 문제

  • 시각적 차별화 부족 — 모든 배너가 크림톤, 같은 구성이라 캐러셀에서 구분 안 됨
  • 클릭 버튼 없음 — 6장 중 2장만 버튼 있음 → 누를 곳이 없다
  • 카피가 설명적 — "~까지, ~를 위한" 구조 반복, 호기심 유발 약함
  • 대구 팝업 종료 — 3/26 종료 → 1장 교체 필요 (짝수 유지)
STRATEGY

혼합 큐레이션 전략

페어별로 다른 고객 의도를 커버. 페어 내에서는 통일감, 페어 간에는 시각적으로 확실히 달라야 넘기기 동기가 생긴다.

PAIR 1 니즈 기반 — "목적이 있는 고객"
Before: 첫 출산준비

"우리 아기 첫 옷장"

메인 카피안
"출산 준비 리스트, 여기서 끝."
서브: 배냇저고리부터 손싸개까지 한 번에 담아요
왜 클릭하나: 출산 준비물 리스트를 따로 안 찾아도 됨
타겟: 예비맘, 출산 선물 찾는 사람
버튼 문구: "한번에 보기 →"
Before: 온가족 패밀리룩

"우리 셋, 같은 옷 입는 날"

메인 카피안
"찍으면 바로 프사감."
서브: 신생아~성인까지, 패밀리 매칭 세트
왜 클릭하나: "프사감" = 결과를 상상하게 함 (SNS 욕구)
타겟: 가족사진, 돌잔치, SNS 업로드
버튼 문구: "매칭 세트 보기 →"
🎨 톤 가이드: 따뜻한 크림/베이지 — 현재 톤 유지, 배너 1은 클로즈업, 배너 2는 가족 풀샷으로 구도 차별화
PAIR 2 발견 기반 — "구경하러 온 고객"
Before: Pick카부

"엄마들이 제일 많이 담은 조합"

메인 카피안
"이번 달 장바구니 1위 조합"
서브: 피카부가 골라낸 베스트 매칭
왜 클릭하나: "다른 엄마들은 뭘 사지?" 궁금함 자극
타겟: 첫 방문, 뭘 사야할지 모르는 고객
버튼 문구: "인기 조합 보기 →"
Before: 신상품 미리보기 (COMING SOON)

"아직 아무도 안 입었어요"

메인 카피안
"먼저 보는 26SS 신상 7종"
서브: 곧 만나요, 피카부의 새로운 이야기
왜 클릭하나: "아무도 안 입었다" = 희소성 + 선점 욕구
타겟: 재구매 고객, 피카부 팬
버튼 문구: "먼저 구경하기 →"
⚠️ 핵심 변경: 흰 천 배경 → 신상 제품 1~2개 티징 이미지 필수
🎨 톤 가이드: 세이지/민트 계열 — PAIR 1과 확실히 다른 톤. 제품 중심 레이아웃, 감성보다 정보 강조
PAIR 3 상황 기반 — "시즌/이벤트 고객"
Before: 우리 아기 마실룩

"오늘 뭐 입지?"

메인 카피안
"머리부터 발끝까지, 원클릭 코디"
서브: 모델 착장 그대로, 세트로 담아두었어요
왜 클릭하나: 매일의 고민 "오늘 뭐 입히지?" 정확히 해결
타겟: 외출 코디 고민하는 엄마
버튼 문구: "코디 세트 보기 →"
🆕 신규 배너 — 대구 팝업 대체 (데이터 기반 확정)

"선물 고민 끝"

메인 카피안
"받는 사람도, 주는 사람도 만족."
서브: 출산선물·돌선물, 세트로 골라뒀어요
왜 클릭하나: "선물 뭐 사지?" 고민 해결 + 세트 구매 유도
타겟: 출산 축하, 돌잔치 선물 찾는 지인/가족
버튼 문구: "선물 세트 보기 →"
구성: 인기 선물 세트 + 기프트백/포장 옵션 연결
선정 근거: 첫 분석에서 패밀리룩(구매율 0.58%)처럼 목적이 명확한 테마가 구매율 최고. 선물 구매 = 목적 명확 + 세트(평균 결제 34,654원) + 상시 수요
🎨 톤 가이드: 샌드/골드 계열 — 활동적이고 밝은 느낌. 아기 풀샷+배경이 있는 라이프스타일 컷

디자이너 브리프 핵심

전체 공통
  • 모든 배너에 클릭 버튼 필수
  • 메인 카피 15자 이내
  • 서브 카피 25자 이내
  • 1000 x 1000 유지
페어 간 차별화
  • PAIR 1: 크림/베이지 (감성)
  • PAIR 2: 세이지/민트 (정보)
  • PAIR 3: 샌드/골드 (라이프스타일)
OPERATION

배너 로테이션 운영 방식

10일 1사이클 — 위치 로테이션

6개 배너를 2~3일마다 페어 순서를 교체. 10일 동안 모든 배너가 모든 위치를 경험 → 위치 편향 없는 순수 성과 비교 가능

일자 첫 화면 (1~2번) 두번째 (3~4번) 세번째 (5~6번)
Day 1~3 PAIR 1 니즈 PAIR 2 발견 PAIR 3 상황
Day 4~6 PAIR 2 발견 PAIR 3 상황 PAIR 1 니즈
Day 7~10 PAIR 3 상황 PAIR 1 니즈 PAIR 2 발견

사이클 후 판단 기준

  1. 방문자 변화 — 보러 오는 사람이 줄고 있는가?
  2. 구매율 — 보고 있지만 안 사는가?
  3. 1인당 탐색 수 — 꼼꼼히 보는가, 바로 나가는가?
  4. 매출 기여 — 실제 매출로 이어지는가?

교체 의사결정

  • 성과 좋음 → 유지, 다음 사이클 앞 순서 배치
  • 많이 보는데 안 삼 → 문구/버튼 수정
  • 보지도 사지도 않음 → 테마 or 이미지 교체
  • 3사이클 연속 부진 → 테마 자체를 교체
