9 앞으로 이렇게

바로 할 수 있는 것

  • 베이직 라인은 단독 피드 대신, 인기 제품과 코디로 보여주기 — "보니 블루머 + 샤인 이너" 세트 착장. 베이직은 주인공이 아니라 좋은 짝꿍 역할
  • 오픈 포스트는 하루에 1개만 — 2개 이상 올리면 서로 묻힘
  • 티저부터 오픈까지 매일 1개 — 중간에 쉬면 관심이 꺼짐
  • 인스타 프로필/스토리 링크에 추적 코드(UTM) 붙이기 — 지금은 인스타에서 얼마나 왔는지 정확히 모름

제품 기획에 참고할 것

  • 신상 중 1개는 반드시 "피카부다운" 베이비 아이템 — 블루머, 세트, 헤어밴드처럼 인스타에 올리고 싶은 디자인
  • 베이직 라인은 인기 제품의 코디 파트너로 기획 — 처음부터 "보니랑 같이 입히면 예뻐요"를 전제로
  • 2~3개 동시 오픈 시, 성격이 다른 제품을 섞기 — 베이직끼리 묶으면 피드 전체가 무난해짐

대표님 확인 필요

  • 에어(4/9)는 어떤 성격의 제품인가? — 귀여운 라인이면 보니처럼 단독 피드, 베이직이면 코디 중심으로 콘텐츠 방향이 달라짐
  • 링고 추가 콘텐츠 투자 여부 — 현재 62,000원/3일. 별도 푸시 vs 코디에서 자연 노출로 충분한지
  • 베이직 라인 콘텐츠 방식 변경 — 기존 "OPEN!" 단독 피드 → 인기 제품과 코디로 보여주는 방식으로 바꿔도 되는지
데이터 출처와 주의사항
- 매출: Cafe24 API (items 합산 방식). 셀러센터 실수치와 약 0.1% 오차 가능. 최종 확정은 셀러센터 기준
- 사이즈 분석: Cafe24 주문 아이템의 옵션 텍스트에서 추출. "기타"(양말, 보넷, 배냇 등 숫자 사이즈가 아닌 제품) 제외 후 계산
- 방문/검색: Google Analytics 4 (GA4). 인스타 인앱 브라우저 유입은 일부 "(not set)"으로 잡혀 인스타 기여가 과소평가될 수 있음
- 인스타: Instagram Graph API v21.0. 4/3 스토리는 24시간 후 만료되어 데이터 없음
- 분석 기간: 4/3(금)~4/5(일) 3일간. 4/5 GA4 데이터는 처리 지연으로 일부 변동 가능
