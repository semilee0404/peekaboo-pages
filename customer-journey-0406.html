2 어떻게 더 많은 사람이 오게 할까

지금 인스타에서 자사몰로 직접 오는 사람이 150명뿐입니다. 12,353명이 봤는데요.
이 150명을 500명, 1,000명으로 늘리는 구체적인 방법들입니다.

실험 1 — 우선 추천

스토리 링크 스티커 + "지금 구경하기" 문구

스토리 링크 스티커 평균 클릭률 0.9%. 피카부 스토리 도달 2,000명이면 약 18명이 클릭. 하루 3개 스토리면 54명. 지금 하루 인스타 유입 50명의 2배.
이렇게 합니다:
1. 스토리 올릴 때 링크 스티커에 상품 URL 넣기 (프로필 링크 유도 X)
2. 스티커 문구를 "지금 구경하기" 또는 "재고 확인하기"로 바꾸기 — 기본 "링크 보기"보다 클릭률 34% 높음
3. UTM 코드 붙이기 (UTM 생성기 활용)
4. 스티커 위치는 화면 하단 중앙 (엄지 터치 영역)
확인 방법: GA4 → 획득 → 소스/매체 → "instagram / story" 유입 수와 구매 확인
목표: 2주 내 인스타 일일 유입 50명 → 150명
참고: Shopify 2026 보고서 — 링크 스티커 CTA 커스텀 시 CTR 34% 향상
실험 2 — 우선 추천

하루 스토리 3~4개, 시간대 분산

스토리 1~7개 사이에서 완료율(끝까지 보는 비율) 87%로 최고. 8개 이상부터 피로감. 시간대를 분산하면 스토리바(프로필 아이콘) 상단 노출 시간이 늘어남.
이렇게 합니다:
1. 오전 9시: 오늘의 제품 소개 or 코디 (링크 스티커)
2. 점심 12시: 투표/퀴즈 ("이번 주 코디 A vs B?") — 참여하면 알고리즘이 다음 스토리도 상위 노출
3. 오후 3시: 실시간 반응 ("오늘 가장 많이 팔린 컬러") (링크 스티커)
4. 저녁 7시: 고객 착장 리그램 or 감성 컷 (링크 스티커)

주의: 한 번에 4개 몰아올리기 X. 시간을 나눠야 스토리바 상단에 계속 뜸
확인 방법: 인스타 인사이트 → 스토리별 도달, 링크 클릭, 답장 수 비교
목표: 스토리 일 도달 2,000 → 3,000+ (시간 분산 효과)
참고: 2026 인스타 벤치마크 — 스토리 1~7개 완료율 87%, 첫 프레임 이탈 23.8%이므로 첫 장이 중요
실험 3

카운트다운 스티커로 오픈일 알림 받기

카운트다운 스티커를 누르면 오픈 시간에 자동 알림이 감. 이메일/푸시보다 열람률이 높음. 기대감을 만들면서 동시에 오픈일에 사람을 자동으로 모음.
에어 오픈(4/9) 적용 예시:
1. 4/6 스토리: 에어 소재 티저 + 카운트다운 스티커 "4/9 오후 2시 오픈"
2. 카운트다운 누른 사람에게 오픈 시간에 자동 알림 전송
3. 4/9 오픈 시: 스토리에 "지금 오픈!" + 링크 스티커

피카부가 3/28 링고 릴스에서 "컬러 퀴즈"로 댓글 23개를 만든 것과 같은 원리. 참여시키면 오픈일에 돌아옴.
확인 방법: 카운트다운 알림 신청 수 → 오픈일 스토리 도달 수 → 자사몰 유입 수 비교
목표: 카운트다운 알림 신청 100명+
실험 4

릴스로 비팔로워 유입 — 새로운 사람에게 닿기

릴스는 팔로워가 아닌 사람에게도 노출됨 (탐색탭). 피카부 팔로워 ~10만명인데, 릴스 하나로 비팔로워 수만 명에게 도달 가능. 3/28 링고 릴스도 6,211도달.
잘 되는 릴스 유형 (아동복 브랜드 기준):
1. "1벌로 3코디" — 같은 옷으로 캐주얼/나들이/데일리 3가지 연출 (15~30초)
2. 컬러 퀴즈/투표 — "이 코디, 몇 가지 컬러일까요?" (링고 릴스처럼)
3. 착장 전후 비교 — 옷걸이 상태 → 아이가 입은 모습 전환
4. 패밀리룩 — 엄마+아기 시밀러룩 (2026년 키즈 패션 트렌드 1위)

주 1~2개면 충분. 릴스는 도달 확장용, 구매 전환은 캐러셀과 스토리가 담당.
확인 방법: 릴스 인사이트 → 비팔로워 도달 비율, 프로필 방문 수, 팔로우 전환
목표: 릴스당 비팔로워 도달 5,000+
참고: 베베드피노(166K 팔로워) — 시즌 드롭 릴스로 비팔로워 도달 극대화, 릴스 도달이 캐러셀의 1.36배
3 어떻게 온 사람이 더 많이 사게 할까

2,464명이 상품을 봤는데 360명만 담았습니다 (15%).
상세페이지를 고치지 않고, 콘텐츠 단계에서 "사야 하는 마음"을 미리 만드는 방법입니다.

핵심 발견: 샤인을 산 사람의 69%가 보니, 쇼콜라, 코이와 같이 샀습니다.
가장 많이 같이 산 조합: 보니 헤어밴드(10건) + 쇼콜라(9건) + 보니 블루머(9건).
이 조합을 상세페이지가 아닌 콘텐츠에서 먼저 보여주면, "이렇게 사야겠다"를 결정한 채로 자사몰에 옵니다.
실험 5 — 핵심 실험

코디 캐러셀 — "이렇게 입히면 이렇게 예뻐요"

캐러셀 평균 저장률이 다른 포맷의 3배. 저장 = "나중에 사야지" = 다음 날 검색 → 구매. 코디 캐러셀은 베이직 제품을 귀여운 제품의 짝꿍으로 보여줘서 교차구매를 유도함.
7~10장 구성:
1장: 후크 — "봄 나들이 코디 5가지" (저장 유도)
2~6장: 각 코디 사진 + 제품명 (예: "보니 블루머 + 샤인 이너")
7장: 전체 제품명 정리
8장: "저장해두고 쇼핑할 때 참고하세요"

실제 적용 예시 (에어 오픈 시):
- "에어 수트 + 보니 헤어밴드 = 나들이 코디"
- "에어 수트 + 코이 블루머 = 집콕 코디"
- 각 코디에 전체 가격 표시 → "이 조합으로 ○○원"

주 1~2개 발행. 이게 피드의 메인 콘텐츠가 되어야 합니다.
확인 방법: 코디 캐러셀 저장 수 → 다음날 사이트 검색량 → 교차구매율 변화
목표: 캐러셀당 저장 50개+, 교차구매 포함 주문 비율 70%+ 유지
참고: 캐러셀 저장률 0.55%(업계 평균), 다른 포맷 대비 95% 높음. 6~13장이 최적 (Social Insider 2026)
실험 6 — 핵심 실험

스토리 투표/퀴즈로 제품 인지 + 참여

투표 스티커 추가 시 인게이지먼트 30% 상승. 참여한 사람에게 알고리즘이 다음 콘텐츠도 우선 노출. 제품명을 자연스럽게 기억시키는 효과.
유형별 예시:

투표 "이번 주 코디: 보니핑크 vs 쇼콜라베이지?"
→ 투표 결과를 다음 스토리에서 공개 + 우승 코디 링크

퀴즈 "이 코디에서 안에 입힌 이너는? ①샤인 ②링고 ③샌디"
→ 맞힌 사람에게 "정답! 지금 보러가기" 링크

이모지 슬라이더 "이 컬러 조합, 얼마나 좋아요?"
→ 컬러 선호도 파악 + 인기 컬러 기반 콘텐츠 제작

질문 스티커 "사이즈 고민 있으신가요?"
→ 답변을 모아서 사이즈 가이드 캐러셀 제작 소재로 활용
확인 방법: 투표/퀴즈 참여율, 참여 후 다음 스토리 도달 변화
목표: 투표 참여 200명+ / 퀴즈 정답률로 제품 인지도 측정
참고: 인스타 2026 알고리즘 — 공유 > 저장 > 댓글 > 좋아요 순 가중치. 인터랙티브 스티커가 이 시그널을 강하게 만듦
실험 7

"품절 임박" 스토리 — 지금 안 사면 없다

베이직 라인은 "나중에 사지" 심리가 강함. "지금 안 사면 없어진다"는 긴급함이 있어야 장바구니에 담김. 실제로 보니 블루머가 대구 팝업에서 빠르게 소진됐을 때 예약주문이 폭발한 패턴.
이렇게 합니다:
1. "80 사이즈 핑크, 마지막 3개" — 실제 재고 기반 (거짓은 X)
2. "오늘 가장 많이 팔린 컬러 TOP 3" — 남들이 사고 있다는 사회적 증거
3. "오픈 3일 만에 ○○개 판매" — 신상 초기 반응 공유

모든 스토리에 링크 스티커.
확인 방법: 품절임박 스토리 올린 날 vs 안 올린 날의 해당 제품 판매 수 비교
목표: 품절임박 스토리 당일 해당 제품 판매 +30%
실험 8

고객 착장 리그램 (UGC)

"실제로 입힌 사진"이 가장 강한 구매 동기. 모델 사진보다 일반 아기가 입은 사진이 "우리 아이도 이렇게" 반응을 만듦. 제작 비용 0원.
이렇게 합니다:
1. 대구 팝업 구매인증 이벤트(~4/5 마감)에서 모인 사진 활용
2. 고객 태그된 스토리/피드를 리그램 (DM으로 허락 받고)
3. 피드에는 "@○○님의 보니 코디" 캐러셀, 스토리에는 수시로
4. 나중에 "Add Yours" 스티커로 "우리 아이 오늘의 코디 공유해주세요" — 자발적 UGC 수집

주의: 사진 퀄리티가 낮아도 괜찮음. "리얼함"이 포인트. 보정 과하면 피카부답지 않음.
확인 방법: UGC 포스트 저장/공유 수 vs 자체 촬영 포스트 비교
목표: 주 1회 이상 UGC 콘텐츠 발행
